# Overview

This is a futuristic arcade racing game built entirely with Gemini 3. It is a cyberpunk-themed 20-level racing game with 10 different vehicles to choose from. The game features nitro boost and a rainy environment, along with the natural road. The game also has its own built-in music along with an option to add custom tracks while playing.

# Key Features in More Details.

### 10 Selectable Vehicles

Each vehicle has unique stats, including:

* Speed

* Grip

* Size

* Vehicle Type (car/bike)

* Special effects (glitch outlines, neon accents)

(The vehicle system is visible in the selection screen grids and stat bars.) 

Players can pick from futuristic machines like:

* Flux Sedan

* Neon Cycle

* Ghost Rider

* Viper GT

* Vector Tank

…and more.

---

### 20 Futuristic Zones

The game progressively increases difficulty across 20 cyberpunk environments, such as:

* NEON BAY

* DATA HIGHWAY

* ACID RAIN

* THE SPRAWL

* FIREWALL

* GRID CORE

* SYSTEM CRASH

* CYBER HELL

Each zone adjusts **speed**, **traffic density**, and optional **rain effects** to increase challenge. 

---

### Nitro System

Players can hold **UP / W** to activate Nitro Boost, which:

* Increases speed by 50%

* Doubles score multiplier

* Adds neon jet effects

* Triggers unique synthesised sound effects

* Consumes nitro fuel, regenerating over time 

---

### Full Audio System

The game supports:

* Built-in background soundtrack (Zen1th – *The Return*)

* A fully working **custom MP3 loader**

* Synth-generated SFX (crash, nitro, tones)

Audio can be muted/unmuted using in-game controls.

---

### Collision & Game Over System

Procedural crash detection handles:

* Bounding-box collisions with dynamic traffic

* Screen shake effects

* Noise-based crash sound

* Game Over UI showing **score** and **level reached**

The player can **Quick Retry** or return to the main menu.

---

### Controls

* **← / A:** Move Left

* **→ / D:** Move Right

* **↑ / W:** Nitro Boost

# Language Used by Gemini 3

* HTML: Structure of game, UI, and canvas

* CSS: Styling, neon theme, effects, layouts

* JavaScript: Game logic, physics, rendering, input, audio

# Installation Process

There is no installation required — the game is built entirely with HTML, CSS, and JavaScript, and runs locally in any modern browser.

Step 1: Download the 'Neon Overdrive V - Hyper Speed (New Edition) [Full Package].zip' file.

Step 2. Extract it.

Step 3: Double-click on the 'Neon_Overdrive_V_Hyper_Speed_New_Edition.html' file to open it in your browser (Chrome, Edge, Firefox, Brave, etc.) and start playing!
