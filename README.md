# Overview

This is a futuristic arcade racing game built entirely with Gemini 3. It is a cyberpunk-themed 20-level racing game with 10 different vehicles to choose from. The game features nitro boost and a rain environment along with natural road. The game also has it's own inbuilt music along with an option to add custom tracks while playing.
